**handle_click(pixel_pos, state, game, selected_cell_ref, game_state_ref, action_result_ref) logic:**

The function modifies selected_cell and triggers a move. Since Python closures over mutable dicts are cleaner than passing refs, use a shared `gs` dict in main() containing: `gs["selected_cell"]`, `gs["game_state"]`, `gs["state"]`, `gs["ai_last_dest"]`.

```python
def handle_click(pixel_pos, gs, game):
    px, py = pixel_pos
    col = (px - BOARD_OFFSET_X) // CELL_SIZE
    row = (py - BOARD_OFFSET_Y) // CELL_SIZE

    if not (0 <= row < 5 and 0 <= col < 5):
        gs["selected_cell"] = None
        return

    state = gs["state"]
    cell_val = state[row, col]
    src = gs["selected_cell"]

    if src is None:
        # Select only player-1 pieces (positive values)
        if cell_val > 0:
            gs["selected_cell"] = (row, col)
        # Clicking joker (0) or opponent piece does nothing
        return

    # A piece is already selected — try to move
    src_idx = src[0] * 5 + src[1]
    dest_idx = row * 5 + col
    valid_moves = game.get_valid_moves(state, 1)

    if dest_idx in valid_moves[src_idx]:
        # Valid move: apply it
        action = [src_idx, dest_idx]
        game.get_next_state(state, action, update_meta_parameters=True)
        gs["selected_cell"] = None
        gs["ai_last_dest"] = None  # clear AI highlight on human move
        # Transition to AI turn (AI_THINKING state) — Plan 02 will handle AI computation.
        # For now, transition directly back to WAITING_FOR_HUMAN (AI is a no-op stub).
        gs["game_state"] = GameState.WAITING_FOR_HUMAN
        # Check terminal
        value, is_terminal = game.get_value_and_terminated(state, action)
        if is_terminal:
            gs["terminal_value"] = value   # 1 = human won, -1 = draw/loss
            gs["game_state"] = GameState.GAME_OVER
    else:
        # Not a valid destination — deselect (no move made, per locked decision)
        gs["selected_cell"] = None
```
